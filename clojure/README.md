# Clojure Restful example

## Get Leiningen

### Manual installation

Follow install instructions on the project site:
[Install](//leiningen.org/#install).

### SDKMAN installation

```
sdk install leiningen
```

## Compile and run

```
lein run
```

## Test performance

```
ab -n 1000 -c 4 http://localhost:3000/
```
